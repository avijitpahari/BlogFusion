# 📝 PHP Blog Management System

🚀 A modern and dynamic **Blog Website** built with **Core PHP & MySQL** featuring a complete **Role-Based Access System (User, Author, Admin)**.

---

## ✨ Features

### 👤 User

* Register & Login system
* Read blog posts
* Comment & like posts
* View profile

### ✍️ Author

* Create, edit & delete posts
* Upload images
* Manage own content

### 🛠️ Admin

* Dashboard with analytics
* Manage users & roles
* Manage posts & categories
* Moderate comments

---

## ⚡ Key Highlights

* 🔐 Role-Based Access Control (RBAC)
* 🔗 Clean URLs using `.htaccess`
* 📌 SEO-friendly slugs
* ♻️ Reusable components (post cards)
* 📱 Fully responsive design

---

## 🏗️ Tech Stack

* **Frontend:** HTML, CSS, Tailwind, JavaScript
* **Backend:** Core PHP
* **Database:** MySQL
